  • 1960년 인터넷 등장 : 분산 통신 시스템, 인프라
  • 1990년 웹 등장 : 스위스 유럽입자물리 연구소(CERN)의 비정규직 연구원 팀 버너스 리가 Enquire 개발(웹의 전신)
    10월 웹페이지 편집기 개발(세계최초)
  • 11월에 WWW(World Wide Web)프로그램 개발(세계최초의 웹브라우저)
  • 12월 웹서버 프로그램 개발 info.cern.ch
1960년에 등장한 인터넷은 엘리트만 향유하는 시스템이었지만 1990년 (원시)웹의 등장으로 인해 본격적인 대중화의 길을 걷기 시작했다.

인터넷이 동작하기 위해서는 2대의 컴퓨터가 인터넷을 통해 송수신 신호를 보냄

  • Web Browser(client): 주소입력 request 화면출력
  • Web Server(sever): 해당파일 response
  1. 자신의 컴퓨터에 웹서버를 둔다 - 대표적인 웹서버 소프트웨어 Apache
    2. IP(Internet Protocol) Address: 127.0.0.1-웹브라우저가 설치되어 있는 컴퓨터를 가르키는 특수한 주소 http:// Hyper Text Transfer Protocol(웹페이지를 전송하기 위해 웹브라우저와 웹서버가 통신하는 규약)
  2. 2. 웹호스팅 업체(웹서버 제공, static, dynamic)에 웹서버를 둔다, 호스트(웹서버가 깔린 컴퓨터) ex) github
